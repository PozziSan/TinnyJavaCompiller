# TinnyJavaCompiller
A tinny Compiller written in Java for the Compiler and Theory of Computing Discipline at University.

# Features:
- Parser and Lexer with the followings Java tokens:
- Package
- Import
- Class
- Const
- Ident
- Arithmetic Operator
- Relacional Operator
- Data Type
- Attribution
- Command Separator
- Arg Separator
- IF
- ELSE
- FOR
- DO
- WHILE
- Modifier

# JDK:
Oracle JDK 8
